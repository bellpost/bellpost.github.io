使用brew安装指定版本node

2020-07-16 | macOS

  2. 2. 安装node@10
  3. 3. 将node@10链接为默认版本

导语： 手贱升级brew update， 导致node升级到14版本，hexo的node使用时总是报错，所以需要退回到node@10版本

查询支持的node版本

使用brew search命令查询版本

可以看到node@10，选择改版本

安装node@10

使用 brew install 安装软件

将node@10链接为默认版本

使用命令brew link，然后根据提示执行，写入默认环境中

这样就可以啦，输入node -v 看下当前版本号吧！
